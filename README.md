## kun
A Simple Inject Library, Which currently only support [JSR330](https://www.jcp.org/en/jsr/detail?id=330) Annotation


### Features
- simple implementation:=
- full support of JSR330
```aidl
    Inject
    Named
    Provider
    Qualifier
    Scope
    Singleton
```
